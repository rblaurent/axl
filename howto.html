How to Use Axl

So you want to get Axl roasting (or hyping) your gameplay? Great choice! Let's get you set up. Don't worry, it's easier than it looks.

Getting Started

First things first - let's get Axl on your computer.

  • 1 Download the installer from the homepage or GitHub releases
  • 2 Run AxlSetup.exe (Windows might show a SmartScreen warning - just click "More info" then "Run anyway")
  • 3 Launch Axl from your desktop and let's get through the setup wizard together

Pro tip: Once installed, Axl auto-updates itself. You'll never have to manually download updates!

First Time Setup

When you launch Axl for the first time, you'll go through a setup wizard. Here's what you need to get ready:

  • ‚Ä¢ Go to Profile Settings and copy your API key
  • ‚Ä¢ Browse the Voice Library, pick a voice you like, and copy the Voice ID

3. Discord Setup

This is the trickiest part, but you got this! We need to create a Discord bot and get some IDs.

Create the bot:
  • ‚Ä¢ Go to Discord Developer Portal
  • ‚Ä¢ Click "New Application" and give it a name
  • ‚Ä¢ Go to the Bot section and click "Add Bot"
  • ‚Ä¢ Copy the bot token (this goes into Axl)

  • ‚Ä¢ Check "Connect" and "Speak" under Bot Permissions

  • ‚Ä¢ In Discord, go to Settings ‚Üí Advanced and enable "Developer Mode"
  • ‚Ä¢ Right-click your server name and select "Copy Server ID"
  • ‚Ä¢ Right-click your voice channel and select "Copy Channel ID"

4. Pick Axl's Personality

This is the fun part! Choose how Axl talks to you:

  • ‚Ä¢ Sarcastic roaster - Classic gaming trash talk and friendly mockery
  • ‚Ä¢ Wholesome hype person - All positive vibes and encouragement
  • ‚Ä¢ Custom - Write your own personality description

5. Set Your Hotkey

Pick a key for instant screenshots when something amazing (or terrible) happens. Default is F8, but you can change it to whatever you like.

Using Axl

Alright, you're all set up! Here's how to actually use Axl:

  • 1 Launch Axl from your desktop
  • 2 Join your Discord voice channel (Axl will connect automatically)
  • 3 Start gaming - Axl will capture your screen every minute and comment when something interesting happens
  • 4 Hit your hotkey (default F8) when something epic happens for instant commentary
  • 5 Check the Console tab in Axl to see what she's thinking

Tips for Best Results

  • ‚úì Make sure you're in the voice channel before starting Axl
  • ‚úì Windowed or borderless windowed mode works best for screen captures
  • ‚úì Choose a personality that matches your vibe and gaming style
  • ‚úì Axl decides when to comment - she won't spam during boring moments
  • ‚úì The more context she has, the better her commentary gets

Customization

Troubleshooting

Running into issues? Here are the most common problems and fixes:

Axl won't join voice channel
  • ‚Ä¢ Make sure the bot is invited to your server
  • ‚Ä¢ Double-check your Server ID and Channel ID are correct
  • ‚Ä¢ Verify the bot has "Connect" and "Speak" permissions
No audio playing
  • ‚Ä¢ Join the voice channel yourself first, then start Axl
  • ‚Ä¢ Check that Axl shows as connected in the UI
  • ‚Ä¢ Look for errors in the Console tab
Screenshots failing
  • ‚Ä¢ Try running Axl as administrator
  • ‚Ä¢ Check Windows screen capture permissions
  • ‚Ä¢ Make sure antivirus isn't blocking screen capture
"API Key Invalid" errors
  • ‚Ä¢ Double-check you copied the full key with no extra spaces
  • ‚Ä¢ Verify you have credits remaining in your API account
  • ‚Ä¢ For Discord token issues, try regenerating it
